# Power Plant

## What's inside?

This Turborepo includes the following packages and apps:

### Apps and Packages

- `api`
- `docs`
- `ui`
- `web`
- `eslint-config-custom`
- `tsconfig`

### Instruction

step 1: Install turbo globally 'npm install -g turbo'
step 2: Install project dependencies 'yarn install'
step 3: create ENV file
'MONGODB_URL=
VITE_SERVER_URL=
SERVER_PORT=
'
step 4: Run yarn dev
